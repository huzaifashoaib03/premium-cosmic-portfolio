## 🔒 Security Architecture (Anti-Inspect & Tamper Shield)

To safeguard proprietary frontend logics and custom engineering frameworks, this repository implements advanced source-code protection measures:

*   **Code Obfuscation:** Variable mangling and control-flow flattening to render scripts completely unreadable to unauthorized scrapers.
*   **Self-Defending Architecture:** Integrated structure anti-tamper checkpoints. Any structural restructuring or formatting (beautification) triggers defensive loops that safely terminate execution.
*   **DevTools Disabling:** Custom event interceptors to safely neutralize mouse right-clicks and developer environment keyboard shortcuts (`F12`, `Ctrl+Shift+I`, `Ctrl+U`).
